# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it via email to **axvi.dev@gmail.com**.

Please do **not** open a public GitHub issue for security vulnerabilities.

We will acknowledge receipt within 48 hours and aim to release a fix as quickly as possible.
